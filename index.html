  • 📅 1989-1990 Quem criou o HTML foi Tim Berners-Lee, um cientista britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear). Ele queria um jeito de compartilhar informações entre os cientistas de forma rápida e acessível via rede.

  • 📅 1991 Tim Berners-Lee publicou o primeiro rascunho do HTML com apenas 18 tags (como title, h1, p, a etc).

  • 📅 1995 Surgiu o HTML 2.0, a primeira versão oficial padronizada.

  • 📅 1997 Veio o HTML 3.2 e depois o HTML 4.01, com mais recursos (frames, tabelas, formulários, etc).

  • 📅 2008-2014 Chegou o HTML5, a versão atual que trouxe: 🎥 Suporte nativo pra vídeo e áudio sem precisar de plugins 📱 Recursos melhores pra dispositivos móveis 🛠️ APIs novas (como Canvas, Geolocation, Web Storage)
